Многофункциональный Telegram-бот, который умеет конвертировать голос в текст, генерировать изображения, собирать тематические подборки из Pinterest и общаться с нейросетями. Разработан как персональный ассистент с разграничением ролей и автоматическими рассылками.

## Возможности

- 🎙 **Голос → текст**  
  Принимает аудиосообщения, преобразует в текст через ffmpeg.

- 🖼 **Генерация изображений**  
  Два режима: через Kandinsky API (Сбер) или локальную DALL-E.

- 📌 **Подборки из Pinterest**  
  По запросу пользователя собирает сетку из 9 картинок на заданные темы (до трёх тем одновременно). Может отправлять автоподборки каждые 4 часа с 8:00 до 20:00.

- 🤖 **Чат с нейросетью**  
  Использует бесплатное API FreeGPT для диалога в личных сообщениях. Бот «общается» как обычный собеседник.

- 🔐 **Ролевая модель и безопасность**  
  Три уровня доступа: обычный пользователь, админ, создатель. Чат с ботом и ответы доступны только доверенным пользователям (middleware-проверка входящих сообщений).

- ⏰ **Автоматические уведомления**  
  - Прогноз погоды на день – каждый день в 6:00.  
  - Расписание занятий техникума на завтра – каждый день с 17:00.

## Технологический стек
- Python 3.10+
- aiogram
- ffmpeg (конвертация аудиофайлов)
- FreeGPT API
- Kandinsky API / DALL-E
- Pinterest API
- BeautifulSoup, Requests, Selenium
- Scheduler (планировщик задач)
